# Andromeda AR

Proyecto web de captación y venta de servicios digitales rentables.

## Qué incluye

- Landing enfocada en conversión.
- Oferta comercial lista para vender automatización, IA y desarrollo web.
- Formulario de leads conectado a API (`/api/leads`).
- Base escalable en Next.js + TypeScript + Tailwind CSS.

## Ejecutar localmente

### Instalación de dependencias

```bash
npm install
```

### Levantar desarrollo

**En Git Bash o Terminal:**
```bash
cd andromeda-ar && npm run dev
```

**En PowerShell:**
```powershell
cd andromeda-ar; npm run dev
```

### Abrir en navegador

```
http://localhost:3000
```

## Próximo paso para rentabilizar

- Conectar el endpoint de leads con tu CRM o WhatsApp API.
- Cerrar ventas con llamada de diagnóstico y propuesta de 3 niveles.
- Escalar con anuncios y contenido orgánico al mismo formulario.
